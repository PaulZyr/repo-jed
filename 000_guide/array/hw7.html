Array HW7

Напишите на JavaScript код, и в нём:
  • Заполните массив таким образом: в первый элемент запишите 'x', во второй 'xx', в третий 'xxx' и так далее.
  • Создайте функцию arrayFill, которая будет заполнять массив заданными значениями. Первым параметром функция принимает значение, которым заполнять массив, а вторым - сколько элементов должно быть в массиве. Пример: arrayFill('x', 5) сделает массив ['x', 'x', 'x', 'x', 'x'].
  • Когда создан массив с числами, узнайте сколько элементов с начала массива нужно сложить, чтобы в сумме получилось больше 10-ти.
